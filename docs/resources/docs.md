## 💡 文档使用建议

1. **优先官方**: 遇到问题首先查阅官方文档,避免被过时博客误导
2. **版本匹配**: 确保文档版本与你使用的库版本一致
3. **API参考**: 开发时将API文档加入书签,随时查阅
4. **示例代码**: 优先运行官方示例,理解后再修改
5. **GitHub Issues**: 文档未覆盖的问题,搜索GitHub Issues寻找答案
6. **社区论坛**: 加入Discord/Slack社区,与其他开发者交流
